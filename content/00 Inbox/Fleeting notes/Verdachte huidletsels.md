# Maligne 
## Pre-maligne huidletsels

|                     | Kenmerken                                                                                                                                                                                                                                | Therapie                                                                                                                                     | Voorbeeld                        |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------- |
| **Actinische keratose** | - roodbruine, vaag omschreven, *schilferende* vlekken<br>- goed voelbaar<br>- meestal multipel, op zonblootgestelde delen<br>- zeer frequent (**1/3 van de oudere patiënten)**<br>- risico op evolutie naar **spinocellulair carcinoom** | - zonprotectie<br>- cryotherapie (beperkte letsels bij niet-risicopatiënt)<br>- chemo/immunotherapie (multipele letsels en/of risicopatiënt) | ![[Verdachte huidletsels.png]]   |
| **Cornu cutaneum**      | - harde hoornkegel op huid<br>- voorloper letsel van **spinocellulair carcinoom**                                                                                                                                                        |                                                                                                                                              | ![[Verdachte huidletsels-1.png]] |
| **Lichen sclerosus**    | - jeukende, witte, parelmoerachtige maculae<br>- vnl anogenitaal<br>- evoluerend naar sclerotische, atrofische huid (phimosis, dyspareunie, obstipatie, verlittekening)<br>- verhoogde kans op ontwikkeling **spinocellulair carcinoom** | - potent corticoïd                                                                                                                           | ![[Verdachte huidletsels-2.png]] |

## In situ maligne huidtumoren

|                  | Kenmerken                                                                                                                                                                                             | Voorbeeld                        |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------- |
| **Ziekte van Bowen** | - intra-epidermaal plaveiselcelcarcinoom huid<br>- goed begrensde, rode tot roodbruine vlek, *met schilfers en korsten bedekt*<br>- vaak solitair<br>- DD [[Psoriasis]], [[Eczeem]], schimmelinfectie | ![[Verdachte huidletsels-3.png]] |
| **Leukoplakie**      | - mucosaal letsel<br>- goedbegrensde, *witte vlek*<br>- vnl op tong, mond of genitalia<br>- CAVE invasief spinocellulair carcinoom                                                                    | ![[Verdachte huidletsels-4.png]] |
| **Lentigo maligna**  | - grillig gevormde, gepigmenteerde macula<br>- sterk wisselende pigmentatie<br>- meestal in gelaat                                                                                                    | ![[Verdachte huidletsels-5.png]] |
