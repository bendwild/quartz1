= Reiter syndroom
= systemische inflammatoire aandoening volgend op een extra-articulaire infectie 
	- [[SOI's]] -> typisch gezien bij een slecht behandelde chlamydia-/gonorroe urethritis
	- gastro-enteritis (salmonella, shigella)

![[image-425.png]]

# Epidemiologie
- 20-40j
- >M
- 70% HLA-B27 positief

# Kliniek
-> typisch 1-6w na infectie
-> **klassieke triade**
	- urethritis
	- [[conjunctivitis]]
	- [[arthritis]]

-> voornamelijk klinische diagnose
-> [[labo]] toont verhoogde inflammatoire parameters

# Behandeling
- [[NSAID]] eerste keuze
- [[antibiotica]] (ev voor nog onderliggende genito-urinaire infectie)

> [!ATTENTION] Verloop
> - meestal gunstig
> - echter zelden cardiac aortitis als complicatie
