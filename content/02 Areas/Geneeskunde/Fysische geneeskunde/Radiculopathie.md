# Radiculopathie
= compressie/irritatie/beschadiging van een **zenuwwortel**

## Etiologie
- [[hernia]]
- stenose
- infectie ([[herpes zoster]])

## Kliniek
- klachten volgens dermatoom
- scherpe uitstralende pijn

## Ischiadicusneuralgie
-> n. peroneus + n. tibialis
- ischiadicussyndroom(chronische, vage rugklachten)  
- lumbago (hevige schietende pijn vanuit rug uitstralend naar been)

→ door druk op n. ischiadicus
- degeneratie tussenwervelschijven (uitpuilen NP)  
- artroseletsels wervelgewrichten (papegaaienbekken)  

### Behandeling
- conservatieve bedrust  
- chirurgisch (zelden nodig, overwegen indien motore uitval)
- zo snel mogelijk mobilisatie & rughygiëne  
**→ parese herstelt vrij vlot, pijn niet alitijd**

# Polyradiculoneuropathie
= schade aan meerdere zenuwwortels & perifere zenuwen
## Etiologie
- auto-immuun ([[Guillain-Barré]], CIDP)
- infectieus

## Kliniek
- symmetrische spierzwakte
- reflexverlies
- autonome stoornissen

## Differentiaal diagnose

|               | [[Guillain-Barré]]                                                                                                       | CIDP                                                              |
| ------------- | ------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------- |
| Omschrijving  | Zeldzame maar potentieel lethale immuungemedieerde polyradiculoneuropathie                                               | Chronische inflammatoire demyeliniserende polyradiculoneuropathie |
| Etiologie     | - onbekend  <br>- vaak na grippaal syndroom of [[vaccinaties]]                                                           | Onbekend                                                          |
| Epidemiologie | **Kan op elke leeftijd!!**                                                                                               | **Meer bij jonge volwassenen**                                    |
| Symptomen     | Slappe (hypotone) verlamming  <br>Areflexie  <br>  <br>**Progressief opstijgend** naar boven → 15% aantasting AH-spieren | Progressieve slappe verlamming  <br>Areflexie                     |
| Lumbaalvocht  | LP: **Albuminocytologische dissociatie** ↔ infectie<br>EMG: afwijkingen aan motorische banen                             |                                                                   |
| Behandeling   | IVIG  <br>Plasmaferese  <br>Tijdelijke respiratoire ondersteuning                                                        | Weinig therapeutische middelen op dit moment                      |
| Prognose      | **>85% herstel op 6m**<br>50+ en [[diarree]] ongunstige prognose                                                         |                                                                   |
