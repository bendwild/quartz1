## Pubertas praecox
borstontwikkeling <8j
testiculair volume >4ml < 9j

#### Meisjes
*veel gevoeliger voor pubertas praecox gezien ovaria heel sensitief zijn voor pulsatiele secreties gonadotrofines -> 5-10x meer dan jongens*

|                  | Centraal                                                                                                                                                                       | Perifeer (=pseudopubertas praecox) |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------- |
| **Idiopathisch** | Premature onset puberteit (70-90%)<br>- geen enkele andere afwijkingen<br>- normale sequentie ontstaan secundaire geslachtskenmerken<br><br>                                   |                                    |
| **Organisch**    | Intracranieel proces (<<<)<br>- abnormale sequentie (dissonant)<br>- gekoppeld met neurologische symptomen, huidafwijkingen...<br>- [[neurofibromatose]]<br>- problemen thv ovaria |                                    |
|                  | ↑ hypofysaire gonadotrofines (LH, FSH) -> ↑ geslachtshormonen                                                                                                                  | ↑ geslachtshormonen                |

#### Jongens
*zeldzaam; testes niet sensitief voor pulsatiele secreties*

|                  | Centraal                                                                                                                                     | Perifeer (=pseudopubertas praecox) |

### Behandeling
- onderliggende oorzaak aanpakken
- GnRH agonist
	- vertraagt pubertaire ontwikkeling -> anders zal kind klein eindigen

### Differentiaal diagnose
- Benigne premature thelarche (geïsoleerde borstontwikkeling)
- Benigne adrenarche (geïsoleerde pubisbeharing)

## Vertraagde puberteit
afwezigheid secundaire geslachtskenmerken >14j (V), >15j (M)

### Etiologie

| Idiopathisch                                                                                              | Nutritioneel                               | Hormonaal                                                                                                                                                                                                                                                                                                                                                                     |
| --------------------------------------------------------------------------------------------------------- | ------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| > jongens<br>- constitutioneel vertraagde [[content/02 Areas/Geneeskunde/Pediatrie/Groei]]<br>- familiaal | - dieet<br>- overmatige fysieke inspanning | *Hypo*gonadotroof [[hypogonadisme]]<br>- constitutioneel<br>- chronische ziekten -> functionele deficiëntie<br>- stoornissen in hyopthalamo-hypofysaire as <br>(panhypopituïtarisme, [[Kallmann syndroom]], hypothyreoïdie...)<br>*Hyper*gonadotroof [[hypogonadisme]]<br>- [[Chromosomale afwijkingen]] -> Klinefelter, Turner<br>- Verworven (na chemo, radio, trauma, operatie...) |

### Behandeling
- onderliggende oorzaak aanpakken
- testosteron of oestradiol om puberteit te versnellen
	- indien idiopathisch
- begeleiding & counseling
