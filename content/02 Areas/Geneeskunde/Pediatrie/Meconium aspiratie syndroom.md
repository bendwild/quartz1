# Achtergrond
Meconium is de eerste stoelgang vd pasgeborene, in utero is er normaal GEEN meconium lozing, maar bij 10% komt dit wel voor. Meconium is visceus en kleverig, dus bij aspiratie kan de baby tijdens de geboorte in de problemen komen (meconiaal aspiratie syndroom). 

# Incidentie
10% heeft meconiaal vruchtwater (>>foetale nood, IUGR, post-terme geboorte)
5% ontwikkelt MAS

# Pathofysiologie
-> **Chemische pneumonitis**
-> **Mechanische obstructie**
# Behandeling
Kinderen met meconiaal vruchtwater -> opgenomen ter observatie voor 24u, want de symptomen kunnen ook pas later optreden.

Kinderen met MAS -> NICU voor kunstbeademing en ventilatie
