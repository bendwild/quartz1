# IUGR
SGA = GGW < P10 voor de [[zwangerschap]]sduur

| **SYMMETRISCH**                                                                             | **ASYMMETRISCH**                                                                            |
| ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| 20-30%<br>Vroeg in de ZS (<18w)                                                             | Late aantasting in de ZS                                                                    |
| GW, L en HO zijn evenredig aangetast (<P10)                                                 | Percentiele GW en L < percentiele HO                                                        |
| Oorzaken<br>- [[Chromosomale afwijkingen]]<br>- Metabole oorzaken<br>- Intra-uteriene infecties | *Utero-placentaire disfunctie*<br>- Maternele [[Pre-eclampsie]]<br>- Tweeling ZS<br>- Roken |
| Meer kans om klein te blijven op latere leeftijd                                            | Vaak goede postnatale catch-up [[content/02 Areas/Geneeskunde/Pediatrie/Groei\|groei]] (niet altijd)                                          |
| Hoge morbiditeit en mortaliteit                                                             |                                                                                             |

## Complicaties

### Foetaal
- intra-uteriene hypoxie en dood (bij slechte monitoring)
- [[asfyxie]]
- hogere kans op [[Meconium aspiratie syndroom]]

### Neonataal
- neonatale [[hypoglycemie]]
	- =/= fysiologisch voorbijgaande [[hypoglycemie]] in eerste 24u door foetaal hyperinsulinisme
	- < geringe hoeveelheid subcutaan vet en leverglycogeen -> hypothermie, [[hypoglycemie]], hypocalcemie
- polycythemie
	- < chronische foetale hypoxie
	- < [[twin-to-twin transfusion syndrome]]
- hypotensie
- NEC
- neurologische sequellen (en mentale  achterstand) op lange termijn

# Macrosomie
= GGW >P90 voor de ZS duur

## Etiologie
- >> **Maternele diabetes**
- Syndromen
	- [[Beckwith-Wiedemann]]: macroglossie (grote tong), microcephalie, umbilicale [[hernia]]
	- [[Sotos]] 

## Complicaties
- >> Risico’s bij de geboorte
	- [[Asfyxie]] (moeilijke partus)
	- Geboortetrauma (schouderdystocie, breuk id clavicula, plexus brachialis letsel, ...)
	- [[Hypoglycemie]]
	- Hypothermie
	- Polycythemie
	- AH moeilijkheden (bij [[Beckwith-Wiedemann]] door macroglossie)
