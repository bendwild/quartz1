# Type 1 - *immediate reaction mediated by IgE antibodies*
= **immediate reaction** within minutes
- [[Allergie]]
- anaphylaxie
- [[Astma]]
- atopie

# Type 2 - *IgG/IgM antibody-mediated cytotoxic reaction*
= reaction occuring in **hours to days**
- haemolytic disease of the newborn
- auto-immune haemoltyic [[Anemie]]
- [[Goodpasture syndrome]]

# Type 3 - *antigen-antibody immune complex-mediated reaction*
= reaction occuring in **hours, days or weeks**
- serum sickness
- [[Rheumatoïde arthritis]]
- SLE
- post-streptococcen glomerulonephritis

# Type 4 - *delayed hypersensitivity*
= **delayed hypersensitivity** -> 24-72u na antigen-exposure
- contact dermatitis
- mantoux test
- **delayed drug reactions!**
