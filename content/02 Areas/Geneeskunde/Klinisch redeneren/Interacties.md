Belangrijkste interacties to remember

# Nierinsufficiëntie
[[NSAID]] + [[diuretica]] + ACE-I -> tripple whammy
[[NSAID]] + anti-hypertensiva -> verminderd anti-hypertensief effect
Ciclosporine -> nefrotoxicisch

# Anti-epileptica
Wees steeds alert bij gebruik van anti-epileptica!
- vele anti-epileptica zijn enzyminducerend
- andere anti-epileptica zijn inhibitoren van specifieke CYP-enzymen
- nauwe therapeutisch-toxische marge

Carbamazepine (BIP) & levonorgestrel -> lagere plasmaconcentratie -> best vervangen door koper-IUD

> [!DANGER] Teratogeniciteit anti-epileptica!! -> altijd [[anticonceptie]] opstarten bij vruchtbare vrouwen

# Therapeutisch-toxische marge
-> geneesmiddelen met nauwe therapeutisch-toxische marge
- VKA
- [[anti-aritmica]] (amiodarone, sotalol...)
- anti-epileptica (carbamazepine, valproaat)
- immunosuppressiva
- clozapine
- levothryoxine
- lithium
- digoxine (-> CAVE combinatie met diuretica en andere geneesmiddelen die kaliëmie verhogen)
	- nausea, diarree en andere gastro-intestinale klachten

# -azoles
Elk -azole (bv fluconazole bij vulvovaginale candida) heeft ander profiel qua inhibitie/metabolisatie door CYP-enzymen -> opzoeken

Fluconazole -> QT-verlenging

# Bloedstolling
Opgelet bij combinaties van: 
- DOAC
- VKA
- acetylsalicylzuur
- [[NSAID]]
- SSRI
- SNRI
- chronisch/overmatig alcoholgebruik

# [[Antibiotica]]

CAVE chinolonen
-> [[ciprofloxacine]]/norfloxacine inhibitoren van CYP1A2
-> mogelijkse QT-verlenging of convulsies in combinatie met andere middelen die QT-verlenging kunnen geven (fluconazole bv)
-> Chinolonen + [[corticosteroïden]] -> sterk verhoogd risico van tendinitis en peesruptuur

 Macroliden
 -> inhibitie CYP3A4 en/of P-gp
 -> QT-verlenging
 
Clarithromycine x rivaroxaban -> versterkte anticoagulatie, verhoogde bloedingsneiging

# Overige 
Methotrexaat -> hepatotoxisch (+ foliumzuur opvolgen)
