# pH
## Respiratoir

|           | Respiratoire acidose                                                                                                              | Respiratoire alkalose                                                                                                          |
| --------- | --------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Etiologie | = hypoventilatie<br>- obstructie<br>- spierzwakte<br>- ARDS<br>- dode ruimte ventilatie<br>- [[sepsis]]<br>- glucoseoverbelasting | = (centrale) hyperventilatie<br>- hypoxemie<br>- salicylaat<br>- XTC<br>- leverfalen<br>- te snelle correctie metabole acidose |
| Kenmerken | - laag pH<br>- hoog pCO2                                                                                                          | - hoog pH<br>- laag pCO2                                                                                                       |
| CAVE      | -> cerebrale VD                                                                                                                   | -> cerebrale VC                                                                                                                |
## Metabool 

|           | Metabole acidose                                                                                                                                                                                          | Metabole alkalose                                                                          |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| Etiologie | - bicarbonaat verlies (GI, renaal, pancreas, gal...)<br>- dalen protonexcretie ([[chronische nierinsufficiëntie]])<br>- accumulatie zuren (verhoogde EW-afbraak, lactaat, ketonen....)                    | - [[Braken]], [[diarree]]<br>- [[diuretica]]<br>- hypercapnie<br>- **hyperaldosteronisme** |
| Kenmerken | - laag pH<br>- laag bicarbonaat                                                                                                                                                                           | - hoog pH<br>- hoog bicarbonaat                                                            |
| CAVE      | - R/ met bicarbonaat geeft verhoogd risico op **intracellulaire acidose** -> enkel te overwegen als **pH <7.15 en symptomatisch, hyperkaliëmie...**<br>- impact acidose op elektrolyten (hyperK, hyperCa) |                                                                                            |
## Gecombineerd

|           | Metabole acidose + respiratoire alkalose | Metabole alkalose + respiratoire acidose |
| --------- | ---------------------------------------- | ---------------------------------------- |
| Kenmerken | - laag bicarbonaat<br>- laag pCO2        | - hoog bicarbonaat<br>- hoog pCO2        |
# Base excess/deficit
= hoeveelheid base die aan 1L bloed moet worden toegevoegd/onttrokken om pH naar 7.4 te brengen (bij pCO2 40mmHG, T 37°C)
= normaal **+2 tot -2 mmol/L**
