= tubulaire dysfunctie door immuun-gemedieerde infiltratie in het interstitium
# Etiologie
- >> medicatie ([[NSAID]], [[antibiotica]], PPI, immunotherapie)
- infectie  (leptospirose, legionella, andere virussen...)
- auto-immuunziekte (SLE, [[sarcoïdose]], [[ziekte van Sjögren]])

# Fysiologie
![[image-296.png]]

**Nefron = glomerulus + tubulus:**
- hoe proximaler, hoe meer bulk gereabsorbeerd wordt (zouten, HCO3-, urinezuur, glucose en kleine EW)

**Beschadiging proximale tubulus**:
- veel HCO3- gaat verloren in de primaire urine => alkalische urinaire pH + renale acidose in het bloed
- veel glucose gaat verloren in de primaire urine, glucosurie heeft 3 oorzakn
	- [[diabetes mellitus]]
	- Fanconi syndroom (syndrome of inadequate reabsorption in the proximal renal tubules)
	- medicatie

**Lis van Henle**
- veel zoutopname

**Distaal**:
- fine tuning van [[kalium]] (distale tubulus) 
- waterabsorptie (distale tubulus en ductus colligens met ADH)

# Kliniek
## Symptomen
Asymptomatisch

Symptomatisch 
-> onset dagen tot weken
- Klachten kaderend in nierfalen (braken, anorexie, algemene malaise)
- Grote gezwollen nieren
- Geeft soms flankpijn
- Bij allergische interstitiële nefritis
    - Koorts
    - Rash
    - Eosinofilie
- **GEEN dysurie en/of pollakisurie**

## Diagnose
[[Labo]]
- oplopend ureum en creatinine
- ev hyperkaliëmie (doe dan ook [[ECG]])

Urinesediment
- typisch **leukocyturie**
- WBC-cilinders (-> dus geen cystitis, komt van tubuli)
- ev RBC

**Urinekweek**
- **negatief** (steriele leukocyturie/pyurie)

Echo
- gezwollen nieren niet obligaat
	< intersititiel infiltraat van immuuncellen met [[oedeem]]

Nierbiopt
- interstitiële infiltraten & -[[oedeem]]
- definitieve diagnose, niet altijd nodig


> [!ATTENTION] Koorts, flankpijn, leukocyturie -> CAVE opstijgende [[Pyelonefritis]] (veel frequenter)
> | INTERSTITIELE NEFRITIS                   | OPSTIJGENDE [[Pyelonefritis]] (BACTERIEEL)                                                                                                                 |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Bilateraal                               | Meestal unilateraal                                                                                                                                    |
| Grote nieren                             | Wigvormig                                                                                                                                              |
| Leukocyturie<br><br>Negatieve urinekweek | Leukocyturie + bacteriurie<br><br>Positieve urinekweek (microbe)                                                                                       |
| Evt koorts                               | Hoge koorts, frissons                                                                                                                                  |
| Geen dysurie/pollakisurie                | Vaak voorafgegaan door cystitis -> dus dysurie, pollakisurie evt in verleden                                                                           |
| AKI                                      | Zelden AKI (omdat het unilateraal is) maar kan stijging serumcreatinine krijgen door hemodynamisch oorzaak: ziek, koorts, braken, nauwelijks gedronken |
| Evt eosinofilie                          | Inflammatoir bloedbeeld uitgesproken                                                                                                                   |
| Nierbiopsie mag                          | CI voor nierbiopt                                                                                                                                      |

# Behandeling
- Stoppen van de verantwoordelijke medicatie
- Behandeling met **[[corticosteroïden]]** om inflammatie thv nier te verminderen
- Ev onderliggende infectie behandelen
