# Pathofysiologie
= lekkage van plasma-EW via de glomeruli *ZONDER uitgesproken ontsteking glomeruli* <-> [[Nefritisch syndroom]]
< weinig selectieve filtration slits + tekort aan podocyten
![[Nefrotisch syndroom pathofysiologie.png]]

# Kliniek
- [[Oedeem]]

# Diagnose
- urineonderzoek
	- *glomerulaire [[proteïnurie]] >3,5 g/d*
	- vroeger 24u-collectie, echter wegens omslachtigheid overgeschakeld naar parameter *[[proteïnurie]]/creatinine concentratie urinair*
		- !! = meetbare [[proteïnurie]] =/= echte [[proteïnurie]]
- bloedonderzoek
	- hypoalbuminemie
	- >> [[hyperlipidemie]]
	- < nierinsufficiëntie
- biopsie
	-> guides treatment

Mnemotic **PLEAC**: 
- **P** rotein
- **L** ipids up
- **E** dema
- **A** lbumin down
- **C** holesterol up

![[image-17.png]]

# Behandeling
Onafhankelijk van onderliggende oorzaak -> *EW-verlies beperken*
1. Inhiberen [[content/02 Areas/Geneeskunde/Nefrologie/RAAS|RAAS]] (ACE-I, ARB, ARNI)
	1. -> verlagen intraglomerulaire druk
2. Zoutbeperking & [[diuretica]]
	2. -> tegen [[oedeem]]
3. Statines 
	3. -> indien hypercholesterolemie

Verder: 
- infecties vermijden/opsporen/behandelen
- thrombose vermijden/opsporen/behandelen
- onderliggende oorzaak aanpakken

# Oorzaken
## Minimale letsel nefropathie
**= meest frequente oorzaak van nefrotisch syndroom bij kinderen!**
- albuminurie
- *geen* hematurie
- normaal serum creatinine, normaal GFR
-> behandelen met *[[corticosteroïden]]* (CAVE vaak herval)
## Focaal segmentale glomerulosclerose
= sclerotische aantasting van gedeelte glomeruli, zowel op macro- als microniveau, kan op alle leeftijden
### Primaire FSGS
- oorzaak onbekend
- behandelen met immuunonderdrukking
### Secundaire FSGS
- [[proteïnurie]], echter vaak geen nefrotisch syndroom
- < [[Obesitas]], HIV
- behandelen met immuunonderdrukking + onderliggende oorzaak aanpakken
### Genetische FSGS
- [[content/02 Areas/Geneeskunde/Nefrologie/RAAS|RAAS]]-blokkers

**CAVE**
- vaak therapieresistent
- frequent evolutie naar nierinsufficiëntie en eindstadium nierfalen
## Membraneuze glomerulopathie
- *geen* hematurie
- verhoogd serum creatinine mogelijk
- op biopt -> subepitheliale neerslagen + verdikte glomerulaire BM
### Primair
= auto-immuun
- AL tegen podocyten 
- immuunonderdrukkende therapie
### Secundair
- medicatie
- [[Hepatitis]] B
- maligniteiten
- -> oorzakelijk behandelen
## Amyloïdose
= stapelingsziekte met overmatige productie en neerslag extracellulaire fibrillen
-> diagnose op biopt waarop amyloid neerslag zichtbaar
-> behandeling oorzakelijk
### AL amyloïdose
- AL = lichte keten
- < hematologische origine
### AA amyloïdose
- AA = amyloid serum A protein = inflammatoir EW
- CAVE verglijdt snel naar [[Nefrotisch syndroom]] en eindstadium nierfalen

![[amyloidose.png|711x929]]

## [[Diabetes mellitus]] nefropathie
= bij *lang bestaande hyperglycemie*
- typisch *10-15j* na start diabetes -> traag progressief, chronisch verloop
	- 1. Hyperfiltratie (met gestegen GFR)
	- 2. Matig verhoogde (micro-)albuminurie
	- 3. Ernstig verhoogde albuminurie
	- 4. [[Nefrotisch syndroom]] + verminderde nierfunctie
	- 5. Eindstadium nierfalen & dialyse
- screening
	- DMT1: vanaf 5j
	- DMT2: vanaf diagnose
- behandeling
	- optimale glycemie regeling & BD behandeling met ACE-I & ARB's
	- Na-Glu cotransporter inhibitor
	- rookstop

![[Diabetes nefropathie.png]]

CAVE 
- bij [[DM II]] is er ook onderliggend vaatlijden dat op zich voor nierlijden kan zorgen
- naast glomerulaire schade, ook *tubulaire schade!* -> tubulo-glomerulaire feedback vormt element van ontstaan van de diabetische nierziekte -> *Na-Glu cotransporter inhibitor*
