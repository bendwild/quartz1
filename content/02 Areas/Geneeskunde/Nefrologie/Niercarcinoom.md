# Maligne tumoren

|               | Renaal cel carcinoom                                                                                                                                                                                                                                                                                                                                                                                                                                               | Nefroblastoom                      | Nierbekken- en uretertumoren                                                                                                                           |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Wat?          | = hypernefroom                                                                                                                                                                                                                                                                                                                                                                                                                                                     |                                    | = transitioneel cel carcinoom (zoals [[blaaskanker]])                                                                                                  |
| Etiologie     | - ROKEN!<br>- [[Obesitas]]<br>- [[hypertensie]]<br>- hereditair: Von-Hippel Lindau, tubereuze sclerose                                                                                                                                                                                                                                                                                                                                                             |                                    | - ROKEN!<br>- fenacetine abusus<br>- chronisce infectie                                                                                                |
| Epidemiologie | - 60-70j piekincidentie<br>- 9de frequentste maligniteit                                                                                                                                                                                                                                                                                                                                                                                                           | - typische nierkanker bij kinderen | - 40-80j<br>- relatief zeldzaam                                                                                                                        |
| Subtypes      | Histologie<br>- clear cell (80-90%)<br>- papillair (10-15%)<br>- chromofoob (4-5%)<br>-> differentiatiegraad wordt uitgedrukt door de Fuhrmanngradatie (**prognostisch zeer belangrijk!**)<br><br>TNM<br>- T1-2: tumoren beperkt tot kapsel<br>- T3-4: tumoren die voorbij kapsel groeien                                                                                                                                                                          |                                    | CAVE ureter en pyelum zeer dunne wand -> snel lokale doorgroei en lymfatische metastsering                                                             |
| Kliniek       | Meestal incidentaloom<br><br>Indien symptomatisch<br>- klassieke triade -> flankpijn, hematurie, palpabele flankmassa<br>- [[B-symptomen]] (koorts, gewichtsverlies)<br>- metastatische symptomen (botpijn, hoesten)                                                                                                                                                                                                                                               |                                    | - hematurie<br>- flankpijn<br>- vermagering<br>- nierinsufficiëntie                                                                                    |
| Diagnose      | - urinestaal (hematurie)<br>- [[labo]] (nierfunctie, [[anemie]]/polycythemie, AF, LDH)<br>- echo (onregelmatige solide massa)<br>- CT contrast (contrast-capterende massa)<br>- staging -> TNM classificatie (**CT abdomen, CT thorax**, ev botscan en CT hersenen)<br>- nierbiopsie (niet standaard, alleen indien chirurgie NIET gepland is, in 15% gevallen inconclusief)                                                                                       |                                    | - retrograde utereografie<br>- **CT**<br>- cystoscopie & cytologie<br>- utererorenoscopie (indien CT inconclusief)                                     |
| Behandeling   | Actieve opvolging<br>- kleine tumoren & grote comorbiditeit<br>**Heelkunde**<br>- partiële nefrectomie laparoscopisch (standaardbehandeling T1)<br>- radicale nefrectomie laparoscopisch (standaardbehandeling T2)<br>- radicale nefrectomie open (standaardbehandeling T3-T4)<br>Ablatie (cryoablatie, RFA)<br>Metastasctomie<br>Radiotherapie (voor bot- en hersenmetastasen)<br>Immunotherapie<br><br>**!! GEEN CHEMO !! -> hypernefromen zijn chemoresistent** |                                    | - radicale nefro-ureterectomie met blaascuff (**gouden standaard**)<br>- distale ureterectomie met ureterreïmplantatie<br>- ablatie<br>- chemotherapie |
| Prognose      | T1: 5j-overleving >90%<br>T3-4: 5j-overleving <50%                                                                                                                                                                                                                                                                                                                                                                                                                 |                                    | T1: >90%<br>T2-T3: 40-75%                                                                                                                              |
