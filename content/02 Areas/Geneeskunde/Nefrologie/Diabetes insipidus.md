# Pathofysiologie

# Etiologie

| Centrale diapetes insipiuds                                                                                     | Nefrogene diabetes insipidus                                                                                 |
| --------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| -> gebrek aan ADH                                                                                               | -> gebrekkige respons aan ADH                                                                                |
| - tumors<br>- trauma<br>- iatrogeen<br>- [[Meningitis]]/[[Encephalitis]]<br>- congenitaal<br>- wolfram syndroom | - medicatie (lithium)<br>- congenitaal<br>- hypercalciëmie<br>- hypokaliëmie<br>- nierziekten (bv [[ADPKD]]) |
![[image-37.png]]
# Kliniek
## Symptomen
- polyurie (>3l)
- polydypsie
- [[Dehydratatie]]
- hypotensie

## Diagnose
### [[Labo]]
- lage urine osmolaliteit
- normaal/hoog serum osmolaliteit

### Dorstproef
**= water deprivation test** -> *is er diabetes insipidus?*
- ADH resistance/insufficiency -> onmogelijkheid tot reabsorptie water -> urine osmolaliteit blijft laag
**= desmopressin stimulation test** -> *wat is de oorzaak van de diabetes insipidus?*
- desmopressine = ADH analoog -> corrigeert centrale insufficiëntie ADH maar niet perifere resistentie

|                                    | **Urine osmolality na dorstproef** | **Urine osmolality**<br>na desmopressine stimulatie test |
| ---------------------------------- | ---------------------------------- | -------------------------------------------------------- |
| **Primary Polydipsia**             | High                               | Not required                                             |
| **Cranial Diabetes Insipidus**     | Low                                | High                                                     |
| **Nephrogenic Diabetes Insipidus** | Low                                | Low                                                      |

# Behandeling
- etiologisch (bv stoppen lithium)
- correctie hypernatriëmie
- desmopressine (centrale DI)
- nefrogene DI moeilijkere aanpak
