|                            | Torsio testis                                                                                                                                                                     | Torsio ovaria                                                                                                                                                                                                                                                                |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Epidemiologie**          | Kan op elke leeftijd, meest frequent:<br>- neonaten<br>- adolescenten<br>- zeldzaam >30j                                                                                          | Voornamelijk vrouwen in de vruchtbare leeftijd                                                                                                                                                                                                                               |
| **Risico-factoren**        | - Bell-clapper deformity<br>- large size<br>- chirurgie in het verleden<br>- plotse verandering in temperatuur<br>- niet-ingedaalde testis<br>- testiculaire pijn in het verleden | - vergrote ovaria >5cm ([[PCOS]], [[zwangerschap]], fertility treatment, ovarian cysts & tumors)<br>- fixatie ovaria vermindert risico ([[endometriose]], maligniteit, adhesies van vorige operaties)<br><br>-> onthoud, **ovarian cysts/tumors are found in >85% of cases** |
| **Kliniek**                | - acute pijn (scrotum, lies, inguinaal)<br>- gezwollen testis<br>- drukpijnlijk<br>- koorts<br>- nausea, braken                                                                   | - acute pijn (lower abdomen, pelvic)<br>- meestal unilateraal met uitstraling<br>- acuut abdomen, soms massa te voelen<br>- nausea, braken<br>- koorts<br>- abnormal vaginal bleeding                                                                                        |
| **Diagnose**               | - cremaster reflex afwezig (niet altijd, soms gewoon wat verminderd)<br>- doppler echo<br>- chirurgische exploratie als torsie niet kan uitgesloten worden                        | - [[labo]] <br>- urine (HCG om [[EUG]] uit te sluiten!!)<br>- doppler transvaginale echo (normale doppler sluit torsie NIET uit)<br>- chirurgische exploratie (laparoscopie/laparotomie) gouden standaard                                                                    |
| **Differentiaal diagnose** | - epididymitis<br>- epididymo-orchitis<br>- scrotaal [[oedeem]]<br>- inguinale [[hernia]]                                                                                             | - [[EUG]]<br>- ruptured ovarian cyst<br>- [[PID]]<br>- [[appendicitis]]<br>- diverticulits<br>- [[UWI]]                                                                                                                                                                      |
| **Behandeling**            | -> manuele detorsie (rotatie van mediaal naar lateraal)<br>-> heelkundige urgentie (**met fixatie bilateraal**)<br>- <6u 90% herstel testis<br>- <12u 50%                         | -> heelkundige urgentie                                                                                                                                                                                                                                                      |
