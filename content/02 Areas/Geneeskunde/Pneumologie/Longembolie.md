# Pathofysiologie
= treedt voornamelijk op als **verwikkeling van een [[DVT]] (40-50% bij patiënten met [[DVT]] OL -> LE)
-> risico-factoren voor veneuze thrombo-embolie (*triade van Virchow*)
- *VENEUZE STASE*
	- atriale fibrillatie
	- immobilisatie (gips, bedlederigheid)
	- heelkunde met >30 mins anaesthesie
- *BESCHADIGING VENENWAND*
	- [[Obesitas]]
	- COVID-19
	- leeftijd >40j
	- fractuur been/majeur trauma
	- varices
	- post-operatief
	- IV-lijnen
- *TOEGENOMEN STOLBAARHEID BLOED*
	- trombofilie
	- [[Zwangerschap]]
	- maligniteit
	- hormonale [[Anticonceptie]]!!
- Voorgeschiedenis VTE

> [!important] Longembolie en [[DVT]] zijn beiden uitingen van 1 pathologie -> *veneuze thrombo-embolie*

# Kliniek
## Symptomen
- [[dyspneu]] (plots ontstaan)
- thoracale pijn
- prikkelhoest
- hemoptoe
- [[palpitaties]], tachycardie
- [[syncope]]

## Klinisch onderzoek
[[Respiratoir onderzoek]]
- tachypneu
- ronchi, crepitaties, pleuraal wrijfgeruis

[[Cardiologisch onderzoek]]
- tachycardie
- gallopritme
- hypotensie
- stuwing v jugularis
- cyanose
- koorts

## Presentatievormen

| Longinfarct (2/3)                                                                       | Onverklaarde [[dyspneu]] (1/4) | Acuut cor pulmonale                                                                                                                         | Silentieus                                                                                                                                                                |
| --------------------------------------------------------------------------------------- | ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| - acute pleurapijn<br>- [[dyspneu]]<br>- hemoptoe<br>- koorts<br>- pleuraal wrijfgeruis |                                | - acute [[dyspneu]]<br>- retrosternale oppressie<br>- cyanose<br>- rechter hart overbelasting<br>- hypotensie<br>- obstructieve shock (10%) | Recurrente silentieuze longembolen kunnen oorzaak zijn van **[[pulmonale hypertensie]]** -> chronische thrombo-embolische pulmonale hypretensie (**CTEPH**) (type IV PHT) |

## Technische onderzoeken
- RX-thorax (normaal bij 50%)
- [[ECG]] (aspecifieke bevindingen)
- Arterieel bloedgas (normaal bij 10%)
	- alkalose
	- hypocapnie (<hyperventilatie)
	- hypoxemie (<V/Q mismatch, verminderd hartdebiet)
- Duplex echografie
	- aantonen [[DVT]]
- **D-dimeren**
	- -> **sensitief** maar **niet specifiek** voor longembolen -> goede uitsluitende kracht
- Perfusie-(ventilatie)scan
	- mismatch
	- CAVE vaak intermediaire probabiliteit
- **CT-scan met pulmonalis angiografie**
	- aangewezen bij vermoeden (massieve) longembolie
- Echocardiografie
	- zal duidelijke afwijkingen tonen bij *massale* longembolie -> ernst LE inschatten

## Risico-stratificatie
Massaal -> >60-70% longcirculatie is afgesloten
Niet-massaal -> <60% longcirculatie is afgesloten, kleine (asymptomatische) longembolen

**Kans op vroegtijdige sterfte?**
- hemodynamische (in-)stabiliteit
- [PESI-score](https://www.mdcalc.com/calc/1304/pulmonary-embolism-severity-index-pesi) (pulmonary embolism severity index -> predicts 30-day outcome of patients with pulmonary embolism)
- RV dysfunctie
- troponine-stijging

# Behandeling
-> NOAC's (rivaroxaban & apixaban)
-> LMWH + NOAC (dabigatran & edoxaban)
-> LMWH + VKA (outdated)
-> LMWH (bij ZS, maligniteit, hoog bloedingsrisico of [[Obesitas]])

![[Pasted image 20250202134944.png]]

Indien *laag* risico op recidief (<3%/j) -> orale anticoagulantia gedurende 3-6m
Indien *hoog* risico op recidief (>8%/j) -> orale anticoagulantia levenslang
