# Spirometrie
-> nuttig voor zowel **restrictief** als **obstructief** [[Longlijden]]

## Statische longvolumes
- VT = teugvolume
- ERV = expiratoir reserve volume
- IRV = inspiratoir reserve volume
- RV = residueel volume
- IC = inspiratoire capaciteit
- FRC = functionele residuele capaciteit
- VC = ERV + VT + IRV = vitale capaciteit
- TLC = VC + RV = totale longcapaciteit
![[Longfunctie-1.png]]
## Dynamische longvolumes
- FEV1 = forced expiratory volume in 1s (=ESW/één seconde waarde)
- FVC = forced vital capacity
- Tiffeneau index = **FEV1/FVC** (>70%) = maat voor luchtstroom beperking
