# Pathofysiologie

> [!INFO] P = Q x R

**R = verhoogde vasculaire weerstand**
- Actief (contractie geactiveerde stellaatcellen) of passief (fibrose, microtrombi..)
- Post/supra/intra/prehepatisch
- Littekenweefsel → etiologie wegnemen

|                     | **Prehepatisch**                                                                               | **Intrahepatisch**                                                             | **Post- of suprahepatisch**                                                                                                                                                               |
| ------------------- | ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Presinusoïdaal**  | - v. porta thrombose<br>- myeloproliferatief syndroom<br>- compressie/invasie door maligniteit | - congenitale leverfibrose<br>- schistosomiasis<br>- primaire biliaire cirrose |                                                                                                                                                                                           |
| **Sinusoïdaal**     |                                                                                                | - [[levercirrose]]<br>- maligniteit<br>- polycystic liver disease              |                                                                                                                                                                                           |
| **Postsinusoïdaal** |                                                                                                | - veno-occlusieve ziekte                                                       | - cardiale problematiek ([[pericarditis]], rechter [[hartfalen]], tricuspidinsufficiëntie)<br>- vv. hepaticae thrombose<br>- web in VCI door oude thrombose<br>- [[Budd-Chiari syndroom]] |
![[image-289.png]]

**Q = verhoogde portale veneuze bloedflow**
→ productie vasodilatoren 
→ splanchische en systemische vasodilatatie 
→ verhoogde flow abdominale arteriën en vena porta

**P = druk in de portale circulatie**
- druk vena porta >5mmHg
- drukgradiënt tussen vena porta en vena cava >5mmHg

# Kliniek
-> voornaamste gevolgen van portale hypertensie zijn: 
- [[ascites]] (met ev pleuratranssudaat / hydrothorax)
- collateralen, [[Gastro-intestinale varices]]

# Diagnose
 -> wedged hepatic venous pressure (sinusoïdale druk)
	- katheter opschuiven tot in tak v. hpeatca
	- >5mmHg = portale hypertensie
	- >12mmHg = **risico op bloeding**

> [!INFO] Maak gebruik van Child-Pugh- & MELD-score
> -> inschatting ernst leverlijden en risico op mortaliteit

# Aanpak
-> aan de weerstand kunnen we famacologisch weinig doen, naast eventueel oorzakelijk agens weghalen (bv stoppen met drinken)
-> aan de splanchnische en systemische vasodilatatie kan er farmacologisch wel iets gedaan worden

# Complicaties
- spontane bacteriële **peritonitis**
	- veelvoorkomende complicatie (30-50% patiënten met [[ascites]])
	- obv translocatie kiem < urinair / GIT
	- dringende behandeling met antibiotic en menselijk albumine
	- **bij minste alarmteken (pt met vage klachten & [[ascites]]) diagnostische paracentese**
- [[hepatorenaal syndroom]]
- variceale bloedingen
- hepatische [[encephalopathie]]
