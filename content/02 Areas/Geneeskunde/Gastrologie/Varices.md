= bloed dat zijn weg vindt langs collateralen

| **Locatie**                                                                     | **Gevolg**                                                                     |
| ------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| **Cardia**                                                                      | Gastro-oesophageale varices<br>− Slokdarmvarices<br><br>− Maagbovenpoolvarices |
| **Anus**                                                                        | − Rectumvarices                                                                |
| **Ligamentum falciforme (para-umbilicale venen)**                               | − Caput medusae<br>![[image-293.png]]                                          |
| **Venen van lever naar diafragma en venen in splenorenaal ligament en omentum** |                                                                                |
| **V. porta**                                                                    | − Spontane splenorenale shunts                                                 |
# Symptomen
- asymptomatisch indien gecompliceerd
- collaterale circulatie op buikwand
- syndroom van Cruveilhier-Baumgarten 
	= belangrijk veneus para-umbilicaal netwerk (caput medusae)
- splenomegalie
	- indien uitgesproken secundair hypersplenisme -> pancytopenie

# Diagnose
- endoscopie
	- slokdarm- en maagvarices
	- volume varices
	- red spots op varices (teken van dreigende bloeding)
	- geassocieerde letsels (mozaïek aspect maagmucosa door hypertensieve gastropathie)
	- duodenum/colonvarices (uitzonderlijk)
- elastometrie (stijfheid lever)
- echografie
- doppler
- anigografie (doorgankelijkheid v porta en collaterale circulatie)
- wedged hepatic venous pressure (sinusoïdale druk)
	- katheter opschuiven tot in tak v. hpeatca
	- >5mmHg = [[Portale hypertensie]]
	- >12mmHg = **risico op bloeding**

# Complicaties
- hepatische [[encephalopathie]]
- [[GI-bloedingen]]
	- primaire preventie bij grote varices of kleine varices met red spots
		-> niet-selectieve **beta-blokkers** (propanolol)
		-> indien ernstig [[astma]], [[COPD]] of insulinedependente [[diabetes mellitus]] -> **ligatuur**
	- secundaire preventie
		-> niet-selectieve beta-blokker
		-> langdurige endoscopische sclerotherapie/ligaturen
		-> levertransplantatie
		-> TIPPS
	- acute bloeding
		-> [[ABCDE]] stabilisatie
		-> vaso-actieve stoffen
		-> [[antibiotica]]
		-> endoscopische therapie (ligaturen)
		-> ballon tamponade of Dennis tube (indien endotherapie niet lukt)
		-> TIPPS (bij Child-B met actieve bloeding / Child C <72u)
		-> heelkunde
