# Epitheliale tumoren
-> **9O%**!!

|                | Borderline tumoren                                                                                                                 | Invasief epitheliaal ovariumcarcinoom                                                                                                                                                                                                            |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Definitie**      | In situ carcinoom **zonder stromale invasie**                                                                                      |                                                                                                                                                                                                                                                  |
| **Epidemiologie**  | - 2/100 000 vrouwen<br>- vrl jonge vrouwen (<40j)<br>- 75% tumoren <25j -> borderline                                              | - 11/100 000 vrouwen<br>- gemiddelde diagnose 63j<br>- 8ste frequentste tumor vrouw, echter 5de qua mortaliteit (late detectie -> silent killer)                                                                                                 |
| **Risicofactoren** |                                                                                                                                    | - nullipariteit<br>- infertiliteit<br>- vroege menarche, late [[menopauze]] (aantal ovulaties)<br>- [[endometriose]]<br>- mutaties **BRCA1/2**, Lynch syndroom (**HNPCC**)<br><br>(orale [[anticonceptie]] beschermend)                          |
| **Subtypes**       | - sereus (vaak bilateraal)<br>- mucineus (meestal unilateraal)<br>- endometrioïd<br>- overige                                      | - sereus (meest frequent) (>2/3 bilaterale aantasting)<br>- mucineus <br>- clear cell                                                                                                                                                            |
| **Kliniek**        |                                                                                                                                    | -> vaak aspecifieke klachten<br>- opgeblazen gevoel<br>- mictie- en stoelgangsklachten<br>- vermagering<br>- vermoeidheid<br>- [[ascites]] (vaak, < peritoneale carcinomatose)<br>- ev pijn door torsie/ruptuur                                  |
| **Diagnose**       |                                                                                                                                    | Biochemie<br>- CA 125<br>- CEA (kan positief zijn bij mucineuze tumoren of eerder GI)<br>- AFP, LDH, beta-HCG (-> kiemceltumoren)<br>- AMH (-> sex cord stromale tumor)<br><br>Gynaecologische echografie<br><br>PET-CT / MR whole body diffusie |
| **Behandeling**    | - **chirurgie** (>laparotomie)<br>- GEEN chemotherapie                                                                             | - chirurgie (redelijk extensief, ovariumcarcinoom is een ziekte van het hele abdomen)<br>- chemotherapie                                                                                                                                         |
| **Follow-up**      | - elke 3m [[gynaecologisch onderzoek]] & CA 125 (peritoneale destructie) eerste 2j<br>- 6-maandelijks tot 5j<br>- nadien jaarlijks | CA 125 goede indicator voor verloop<br>- elke 3m klinisch onderzoek en CA 125 (& CEA bij mucineuze tumoren)<br>- 6-maandelijks tot 5j<br>- nadien jaarlijks                                                                                      |
| **Prognose**       | - gunstig -> globale overleving 95%<br>- afhankelijk van stadium bij diagnose                                                      | Meestal overlijden door darmobstructie en asthenie                                                                                                                                                                                               |

> [!DANGER] Silent killer
> Ovariumcarcinoom disproportioneel grote oorzaak van kanker-gerelateerde mortaliteit, waarom?
> -> late diagnostiek
> -> laattijdige en vage klachten
> -> ovariumcarcinoom wordt in 75% van de gevallen in stadium III-IV ontdekt

> - direct ([[Uterus]], rectum, blaas)
> - peritoneaal (hele abdomen)
> - lymfogeen (para-aortaal)
> - hematogeen (laat -> lever, longn bij mucineus/clear cell)
