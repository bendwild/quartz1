# Benigne ovariële massa

| **Type**                                                          | **Kenmerken**                                                                                                                                              | **Symptomen**                                                                                                  | **Beleid**                                                                                                  |
| ----------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **Functionele cysten**                                            | - premenopausaal<br>- verdwijnen spontaan                                                                                                                  |                                                                                                                |                                                                                                             |
| **Follikelcyste**                                                 | - rijpende follikel ovuleert niet en blijft groeien                                                                                                        | - vaak asymptomatisch<br>- soms pijn bij ruptuur/torsie                                                        | - conservatief, controle na 3 maand<br>- orale [[anticonceptie]] kan cyste vorming in de toekomst vermijden |
| **Corpus luteumcyste**                                            | Na ovulatie ontstaat een bloeding in corpus luteum = hemorragische cyste<br><br>=> Progesteron ↑  <br>=> Endometrium dikte is passend (luteaal)            | - vaak asymptomatisch<br>- doffe unilaterale onderbuikspijn<br>- uitgestelde menses met daarna hevige bloeding | - conservatief<br>- heelkunde bij hemoperitoneum                                                            |
| **Organische cysten**                                             |                                                                                                                                                            |                                                                                                                | - verdwijnen niet spontaan, vaak chirurgie nodig                                                            |
| **Endometrioom <br>(= endometriosecyste)<br>(= chocolade cyste)** | - obv [[endometriose]], door opstapeling van oud bloed in het ovarium<br>- bij **postmenopauzale vrouwen** => verhoogd **risico op een maligniteit**       | - chronische pijn, [[dysmenorroe]]<br>- inhoud van cyste => chocoladeachtig bruin vocht<br>                    | - meestal heelkunde (afhv leeftijd & maligniteitsrisico)                                                    |
| **Matuur teratoom <br>(= dermoïdcyste)**                          | - **kiemceltumor** (bevat weefsels uit meerdere kiemlagen -> vet, haar, talg...)<br>- specifieke variant: **struma ovarii** (SK weefsel dominant aanwezig) | - vaak asymptomatisch<br>- bij ruptuur talg => risico op chemische peritonitis                                 | - heelkunde zonder spilling (om peritonitis te voorkomen)                                                   |
| **Sereus cystadenoom**                                            | - epitheliale tumor, meestal **uniloculair** (één enkele holte met vocht)                                                                                  | - vaak asymptomatisch <br>- buikklachten<br>- cyclusstoornissen                                                | - heelkunde, overweeg appendectomie bij een multiloculair mucineus cystadenoom                              |
| **Mucineus cystadenoom**                                          | **Grotere** epitheliale tumor, meestal **multiloculair** (bestaat uit meerdere met elkaar verbonden holtes, gescheiden door tussenschotte)                 |                                                                                                                |                                                                                                             |

# Benigne niet-ovariële massa

| **Type**                              | **Kenmerken**                                                                                               | **Symptomen**                                                           | **Beleid**                                                                |
| ------------------------------------- | ----------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| **Para-ovariële / paratubaire cyste** | - ontstaan uit embryologische resten van de buizen van Wolff                                                | - meestal asymptomatisch                                                | - enkel heelkunde indien symptomatisch                                    |
| **Hydrosalpinx**                      | - obstructie tuba (postinfectieus/chirurgie) met vochtophoping                                              | - meestal asymptomatisch<br>- soms vaginaal verlies<br>- milde buikpijn | - heelkunde bij fertiliteitsproblematiek                                  |
| **Tubo-ovarieel abces**               | - ontsteking ovarium + tuba door [[PID]]<br>- vnl bij premenopausale vrouwen                                | Infectietekenen<br>- koorts & malaise<br>- buikpijn                     | - [[antibiotica]]<br>- heelkunde indien GEEN beterschap                   |
| **[[EUG]]**                           | - implantatie buiten de baarmoeder <br>- denk er aan bij een tubaire zwelling & lege [[Uterus]] -> hCG test | - amenorroe<br>- unilaterale pijn<br>- spotting                         | Afhankelijk van hCG en klachten<br>- conservatief<br>- MTX<br>- heelkunde |
| **Pseudocysten**                      | - vocht tussen adhesies (postoperatief/infectie)                                                            | - vaak asymptomatisch                                                   | - conservatief                                                            |
