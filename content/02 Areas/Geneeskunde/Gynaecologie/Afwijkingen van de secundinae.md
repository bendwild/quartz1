# Vruchtwater

| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Etiologie    | - [[(P)PROM]]<br>- placentaire insufficiëntie<br>- postmaturiteit<br>- [[Potter syndroom]]<br>- [[Twin-to-twin transfusion syndrome]]                                                                                                                                                                       | Materneel<br>- slecht gecontroleerde [[diabetes mellitus]]<br><br>Foetaal<br>- GIT-obstructies (atresie, compressie)<br>- slikstoornissen<br>- foetale polyurie ([[Twin-to-twin transfusion syndrome]])<br>- [[hartfalen]]                                                                                                                                                                                      | - verhoogde maternale leeftijd<br>- inductie [[Bevalling]]<br>- sectio<br>- meervoudige [[zwangerschap]]                                                                                                                                                    |
| Diagnose     | - echo (meting vruchtwater + nakijken foetale anatomie)<br>- PROM-test (opzoeken van occult vruchtwaterverlies)<br>- Doppler + CTG (evaluatie foetaal welzijn bij IUGR)                                                                                                                                     | - echo (meting vruchtwater + nakijken foetale anatomie)<br>- testen maternale DM                                                                                                                                                                                                                                                                                                                                | Kan zich presenteren als [[sepsis]], [[longembool]] of anafylaxie,met een plotseling begin van volgende symptomen: <br>- [[dyspneu]], hypoxie<br>- hypotensie<br>- coagulopathie ([[DIC]])<br>- tachycardie<br>- verwardheid, convulsies<br>- hartstilstand |
| Complicaties | Materneel<br>- verhoogde kans op sectio (<foetale stress, IUGR, navelstrengcompressie)<br>- [[(P)PROM]] geeft verhoogd risico op chorio-amnionitis<br><br>Foetaal<br>- pulmonaire hypoplasie<br>- IUGR -> meer risico op hypoxische hersenschade<br>- [[(P)PROM]] geeft meer risico op neonatale [[sepsis]] | Materneel<br>- uteriene overdistensie -> vroegtijdige contracties, [[prematuriteit]], abruptio placentae, postpartum hemorragie<br>- compromittering respiratoire functie<br>- verhoogd risico op instabiele ligging -> sectio, navelstrengprolaps<br>- te snelle evacuatie vruchtwater -> **abruptio placentae**<br><br>Foetaal<br>- [[prematuriteit]]<br>- hypoxie bij navelstrengprolaps, abruptio placentae |                                                                                                                                                                                                                                                             |
| Behandeling  | -> afhankelijk van oorzaak<br>- [[(P)PROM]] ([[antibiotica]], longrijpen, indien mogelijk inductie rond 37w)<br>- IUGR (foetale bewaking, inductie/sectio)                                                                                                                                                  | -> afhankelijk van oorzaak<br>- evacuerende amniocentese indien ernstige [[prematuriteit]] en ernstig maternaal discomfort (CAVE helpt maar enkele dagen)                                                                                                                                                                                                                                                       | -> voornamelijk ondersteunend (er bestaan geen specifieke behandelingen)<br>-> **MEDISCH NOODGEVAL**                                                                                                                                                        |
