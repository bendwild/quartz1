# Evaluatie
[[Oftalmologisch onderzoek]]
- oogbewegingen & corneareflex
- pupilgrootte & -reactiviteit
	- pinpoint pupillen -> intoxicatie, hersenstamproblemen
	- brede pupillen -> cocaïne, anticholinergica
	- anisocorie -> inklemming
- fundoscopie 
	- papiloedeem -> intracerebrale overdruk

Pijnreactie
- duwen thv orbitarand, nagelbed, sternum, buitenkant oor
- nagaan of er een hemibeeld is

Geëvoceerde potentialen
- SSEP (somato-sensorisch-geëvoceerde potentialen) -> test sensibele banen
- BERA (brainstem evoked response audiometry) -> test auditieve banen

> [!important]  
> SSEP beste prognostische factor bij coma

[Glasgow Coma Scale](https://www.mdcalc.com/calc/64/glasgow-coma-scale-score-gcs) (GCS)
1. Ogen openen (1-4)
2. Verbale respons (1-5)
3. Motorische respons (1-6)

> [!INFO] Minimale score = 3 (niet 0!)
