# Pathofysiologie
- Zona Glomerulosa -> **mineralocorticoids** 
	- aldosterone [[content/02 Areas/Geneeskunde/Nefrologie/RAAS|RAAS]]
- Zona Fasciculata -> **glucocorticoids** 
	- <hypothalamic-pituitary-adrenal axis (CRH -> ACTH -> cortisol)
	![[image-41.png|454x394]]
- Zona Reticularis -> **androgenen**
	- produceert dehydroepiandrosterone (DHEA)
	![[image-42.png|504x276]]

> [!HINT] Mnemonic [[Bijniercortex]]
> _Go Find Rex, Make Good Sex_: 
> - **G** - glomerulosa 
> - **F** - fasciculata
> - **R** - reticularis
> - **M** - mineralocorticoids
> - **G** - glucocorticoids
> - **S** - sex hormones
