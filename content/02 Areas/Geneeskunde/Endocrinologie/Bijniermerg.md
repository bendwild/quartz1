# Pathofysiologie
-> productie van **catecholamines**: 
- adrenaline
- 10% v/h noradrenaline
	- rest komt uit sympathische zenuwstelsel
- dopamine

> [!INFO] Adrenaline en zijn precursoren worden in het lichaam gemetaboliseerd tot **metanefrines** en **vanillylamandelzuur** (VMA)

# Differentiaal diagnose

|               | Hypofunctie bijniermerg                                                                                                 | Hyperfunctie bijniermerg<br>                                                                                                                                                                                                                                                                   |
| ------------- | ----------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Etiologie     | -> degeneratie orthosympatisch systeem<br>- idiopathisch<br>- multi-systeem<br>- verwikkeling van [[Diabetes mellitus]] | -> tumor van het bijniermerg (= **feochromocytoma**) of ander sympathisch weefsel (= **paraganglioom**)                                                                                                                                                                                        |
| Epidemiologie |                                                                                                                         | Zeldzaam (0.8/100 000)<br>-> frequent associatie met andere endocriene tumoren (MEN II, [[neurofibromatose]], Von Hipple Lindau)<br>-> geïsoleerd (sporadisch & familiaal -> genetica)                                                                                                         |
| Kliniek       | - orthostatische hypotensie                                                                                             | - opstoten [[Hypertensie]] met bleekheid, [[palpitaties]], [[Hoofdpijn]], zweten, nervositas (5 P's)<br>- vaak abdominale pijn & angstgevoel<br>- soms orthostatische hypotensie tussen aanvallen                                                                                              |
| Diagnose      | Dynamische test plasma noradrenaline spiegel                                                                            | 1. 24u urinaire excretie catecholamines en metabolieten (VMA)<br>2. Plasmadosage catecholamines<br>	- meer specifiek -> vrij plasma metanefrines<br>3. Chromografine A (in plasma)<br>	- alternatieve marker voor functionele activiteit paragangliomen met normale preoperatieve metanefrines |
| Behandeling   | PO 9-alfa-fludrocortisone 0.1 mg/d                                                                                      | - heelkundige resectie na grondige voorbereiding met IV vocht, zoutsuppletie en alfa-lytica -> hypertensieve crisis tijdens operatie voorkomen                                                                                                                                                 |
| Prognose      |                                                                                                                         | DD tussen maligne/benigne feochromocytoma kan histologisch NIET gemaakt worden<br>-> **minstens 10j follow-up** (bepaling metanefrines en/of chromogranine A)<br>-> levenslange opvolging bij diagnose <2Oj, genetische afwijking, >15cm, paragangliomen<br>-> bij recidief: tot 40M maligne   |
| CAVE          |                                                                                                                         | - Beta-blokker bij feochromocytoma kan paradoxaal hypertensieve opstoot geven<br>- Enkel [[Metastasen]] zijn bewijzend voor maligniteit                                                                                                                                                        |

> [!EXAMPLE] Kliniek HYPERfunctie bijniermerg -> **5P's**
> - **P**ressure ([[Hypertensie]])
> - **P**alpitations
> - **P**allor (pale skin <vasoconstrictie)
> - **P**erspiration
> - **P**ain ([[Hoofdpijn]], abdominaal)
