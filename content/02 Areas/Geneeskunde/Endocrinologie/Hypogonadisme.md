# Overige ziektebeelden
## Gynaecomastie
-> relatief overwicht oestrogeen tov relatieve daling testosteron
- [[content/02 Areas/Geneeskunde/Pediatrie/Puberteit|puberteit]] (fysiologisch)
- chronisch leverlijden
- [[testistumor]]
- Klinefelter
- hyperthyroidie
- idiopathisch

-> discreet schijfje - volontwikkelde vrouwenborst

-> brede onderzoeken
- testispalpatie
- scrotale echo
- hCG
- indien unilateraal -> uitsluiten [[borstkanker]]

-> behandel onderliggende pathologie indien aanwezig
-> anders
- anti-oestrogenen (tamoxifen)
- evt chirurgie

## Hirsutisme
-> abnormaal toegenomen mannelijk beharingspatroon bij vrouw, [[Acne]], androgene alopecia
- [[PCOS]]
- onderliggende endocriene aandoening
- idiopathisch

-> behandel onderliggende oorzaak
- orale contraceptie
- bij onvoldoende effect -> extra anti-androgenen
