| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Wat?        | = chronisch gelokaliseerde vorm van prurigo<br>(geen echte vorm van lichen)                                                                                                             | = echte vorm van lichen (korstmossen)                                                                                                                                                                                                                                                                                                             |                                                                                                                                                                                                                                |
| Etiologie   | < door krabben tgv [[jeuk]]                                                                                                                                                             | < **immuunreactie** van cytotoxische T-lymfocyten tegen antigenen in basale keratinocyten<br>**< associatie met [[hepatitis]] C!**<br>< lichenoïde fase bij graft-vs-host-disease na beenmergtransplantatie<br>< medicatie (beta-blokkers, penicillamine, goudzouten...)                                                                          |                                                                                                                                                                                                                                |
| Kenmerken   | - hevige jeukende, omschreven, matig scherp begrensde plaque<br>- confluerende papels<br>- **rood, schilferend en gelichenificeerd**<br>- wordt onderhouden door patiënt (krabben zelf) | - jeukende, vlakke, glanzende papels<br>- kleur varieert van rood-roodblauw-paars<br>- wickham striae (netwerk van fijnvertakte wittige streepjes aan oppervlak)<br>- meestal symmetrisch gegroepeerd<br>- **ziekte van de 6 p's**: papula, plaque, plat, polygonaal, paarsachtig, pruritus<br>- uitgesproken post-inflammatoire hyperpigmentatie | - kan op jonge leeftijd beginnen<br>- jeukende witte, parelmoerachtige maculae<br>- evoluerend naar sclerotische, atrofische huid (CAVE fimosis, dyspareunie, obstipatie)<br>- verhoogd risico op **spinocellulair carcinoom** |
| Locatie     | -> overal waar patiënt kan krabben                                                                                                                                                      | - tong (grijzige plaques)<br>- buccale mucosa<br>- lippen<br>- nagelaantasting<br>- acuut veralgmeend                                                                                                                                                                                                                                             | -> vooral **anogenitaal**                                                                                                                                                                                                      |
| Behandeling | - **[[jeuk]]-krab cyclus onderbreken**<br>- sterk lokaal corticoïd & antihistaminicum per os                                                                                            | - topische corticoïden (klasse III-IV)<br>- calcineurine remmers (tacrolimus, pimecrolimus)<br>- PUVA/UVB-therapie (ev met retinoïden of ciclosporine)                                                                                                                                                                                            | - potent corticoïd                                                                                                                                                                                                             |
