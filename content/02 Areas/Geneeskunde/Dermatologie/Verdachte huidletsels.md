## Pre-maligne huidletsels

|                         | Kenmerken                                                                                                                                                                                                                                |                                                    | Therapie                                                                                                                                     | Voorbeeld                        |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------- |
| **Actinische keratose** | = goedaardige tumor bestaande uit vetweefsel<br>- roodbruine, vaag omschreven, *schilferende* vlekken<br>- goed voelbaar<br>- meestal multipel, op zonblootgestelde delen<br>- risico op evolutie naar **spinocellulair carcinoom**      | -  zeer frequent (**1/3 van de oudere patiënten)** | - zonprotectie<br>- cryotherapie (beperkte letsels bij niet-risicopatiënt)<br>- chemo/immunotherapie (multipele letsels en/of risicopatiënt) | ![[Verdachte huidletsels.png]]   |
| **Cornu cutaneum**      | - harde hoornkegel op huid<br>- voorloper letsel van **spinocellulair carcinoom**                                                                                                                                                        |                                                    |                                                                                                                                              | ![[Verdachte huidletsels-1.png]] |
| **[[Lichen]] sclerosus**    | - jeukende, witte, parelmoerachtige maculae<br>- vnl anogenitaal<br>- evoluerend naar sclerotische, atrofische huid (phimosis, dyspareunie, obstipatie, verlittekening)<br>- verhoogde kans op ontwikkeling **spinocellulair carcinoom** | - kan op jonge lefetijd beginnen                   | - potent corticoïd                                                                                                                           | ![[Verdachte huidletsels-2.png]] |
