|                   | **Hodgkin lymfoom (10–15%)**                                                                                                                                                                                                                                        | **Non-Hodgkin lymfoom (80–85%)**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Wat?**          | = Inflammatoire opzetting met fibrose rond **Reed-Sternberg cellen** (2 nucleolen) of **Hodgkin cellen** (1 nucleool)                                                                                                                                               | = Maligne tumoren van lymfoïd weefsel, **zonder** Reed-Sternberg cellen<br>= Umbrella term voor >60 types                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **Etiologie**     | -> Mutatie B-lymfocyten tot "large, multi-nucleated cells"<br>-> Vaak door **anti-EBV immuunrespons** (>50%)                                                                                                                                                        | -> 70% B-cel origine, 30% T-cel<br>-> Vaak door **chromosoomtranslocaties**<br>-> Risicofactoren: immuundeficiëntie, auto-immuunziekten, infecties, chemische stoffen, straling                                                                                                                                                                                                                                                                                                                                                                                          |
| **Epidemiologie** | - 2/100 000/j<br>- M/V = 2/1<br>- **Bimodale distributie**:<br>&emsp;• 20–30j (piek)<br>&emsp;• 50–60j (oplopende incidentie)                                                                                                                                       | - 7de meest voorkomende kanker<br>- Mediane leeftijd: 60–70j                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Subtypes**      | Klassiek HL<br>  - Nodulaire sclerose (70%)<br>  - Gemengd cellulair (25%)<br>  - Lymphocyte-rich (5%)<br>  - Lymphocyte-depleted (<1%)<br>Nodulair lymphocyte-predominant (NPHL)                                                                                   | **2 hoofdtypes**:<br>1. *Indolent* (bv. folliculair, marginale zone, lymfoplasmocytair)<br>2. *Agressief* (bv. diffuus grootcellig NHL, **Burkitt lymfoom**)                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Kliniek**       | - Lymfeklieropzetting (vaak 1 halsklierzwelling, klassiek pijnlijk bij alcoholinname)<br>- Febris e.c.i<br>- [[B-symptomen]] (25%)<br>- Organomegalie<br>- Compressiesymptomen (trachea, [[VCSS]])<br>- Zeldzaam: long, pleura, bot, CNS, huid (paraneoplastisch)   | Zeer variabel:<br>- Lymfeklieropzetting<br>- [[B-symptomen]]<br>- Organomegalie<br>- Extranodale aantasting<br>- Merginvasie: [[Anemie]], [[Thrombopenie]]<br><br>Specifiek:<br>- Huidlymfomen: mycosis fungoides, Sézary<br>- MALT: abdominale klachten<br>- CNS: neurologische symptomen                                                                                                                                                                                                                                                                               |
| **Diagnose**      | - Biopt (liefst chirurgisch)<br>- CT thorax-abdomen / PET-CT (**gouden standaard**)<br>- [[Labo]]: inflammatie, soms immuno afwijkingen<br>- Stagering: **[Ann-Arbor](https://radiopaedia.org/articles/ann-arbor-staging-system-historical)**                       | - [[Labo]]: vaak normaal, soms [[Anemie]], [[Thrombopenie]]<br>- Gestegen LDH (typisch zeer hoog bij Burkitt, tumor lysis syndroom), β2-microglobuline<br>- Beenmergpunctie: invasie (20–50%)<br>- Botbiopt: typering NHL, gericht door CT of echografie<br>- **CT thorax-abdomen / PET-CT (gouden standaard**, bewezen waarde bij agressieve NHL en Hodgkin, minder bij indolente NHL)<br>- Lumbaalpunctie (bij agressief NHL, hersenlocalisatie? / invasie CZS?)<br>- Stagering: **[Ann-Arbor](https://radiopaedia.org/articles/ann-arbor-staging-system-historical)** |
| **Behandeling**   | Vroeg stadium (IA–IIA):<br>- Chemotherapie (ABVD) + radiotherapie<br>Gevorderd (IIB–IVB):<br>- Polychemotherapie<br>Recidief:<br>- Intensifiëren chemo<br>- Autologe stamceltransplantatie<br>- Checkpoint-inhibitoren                                              | Combinatie chemotherapie / radiotherapie / anti-CD20 antilichamen<br>- indolente NHL -> niet curatieve behandeling<br>- agressieve NHL -> curatieve behandeling                                                                                                                                                                                                                                                                                                                                                                                                          |
| **Prognose**      | - Stadium I–II zonder risicofactoren: >90% ziektevrije 5-jaarsoverleving<br>- Met risicofactoren: ~80%<br>- Stadium III–IV: ~80%<br><br>Ongunstige factoren:<br>- Bulky disease (>10cm)<br>- [[Anemie]], hypoalbuminemie<br>- Lymfopenie, leukocytose<br>- Leeftijd | Obv **Internationaal Prognostische Index** (IPI)<br>- indolente lymfomen -> genezen niet maar overleven vaak lang<br>- agressieve lymfomen -> genezen volledig of hervallen en/of overlijden binnen 2jaar<br>                                                                                                                                                                                                                                                                                                                                                            |
| **CAVE**          | Late complicaties behandeling:<br>- Chemo: MDS, AML<br>- Radio: carcinomen, atheromatose, longfibrose, hypothyreoïdie<br>Vergeet [[Vaccinaties]] niet                                                                                                               | Niet gespecificeerd                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |

> [!ATTENTION] [[B-symptomen]] 
>  [[B-symptomen]] zijn een klinische uiting van een **chronische inflammatoire respons**
>  - onverklaarde koorts >38 °C, soms piekend = Pell-Epstein koorts
>  - nachtelijk zweten
>  - vermagering (>10% op 6m)

# Differentiatie

| **Kenmerk**              | **Hodgkin’s**                                                                                                    | **Non-Hodgkin’s**                                                   |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| **Verspreiding**         | - contiguous (spreads to adjacent lymph nodes)<br>- rarely involves extra-nodal sites                            | - non-contiguous<br>- multi-centric                                 |
| **Cellen**               | Reed-Sternberg cells (B-cellen)                                                                                  | Geen Reed-Sternberg cells<br>-> B- en T-cellen                      |
| **Leeftijd bij aanvang** | Bimodale distributie<br>- 20-30j (piek)<br>- 50-60j<br><br>Soms ook bij kinderen                                 | Oudere leeftijd (60-70j)<br><br>Zelden <5j, eerder bij adolescenten |
| **Epidemiologie**        | Zeldzaam                                                                                                         | 7de meest voorkomende kanker                                        |
| **Majeure types**        | - klassiek HL<br>- nodulair lymphocyte-predominant HL (NPHL)                                                     | - indolent NHL<br>- agressief NHL                                   |
| **Subtypes**             | - Nodulaire sclerose (70%)<br>- Gemengd cellulair (25%)<br>- Lymphocyte-rich (5%)<br>- Lymphocyte-depleted (<1%) | 60+ types                                                           |

> [!EXAMPLE] Indien lymfeklieropzetting, denk aan:

# CAVE

> [!DANGER] Bij kinderen zijn **ALLE Non-Hodgkin lymfomen agressief**
> - burkitt lymfoom en lymfoblasten lymfoom meest frequent

> [!DANGER] Mediastinaal Non-Hodgkin lymfoom = **URGENTIE**
> - bloedbeeld
> - beenmerg
> - beeldvorming (trachea diameter?)
> - reageren vaak wel goed op therapie
