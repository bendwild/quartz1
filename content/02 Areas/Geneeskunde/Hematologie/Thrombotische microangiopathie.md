= groep van aandoeningen gekenmerkt door de *obstructie van de microcirculatie* door trombi 

![[Thrombotische microangiopathie.png]]

> [!important] Belangrijkste vorm bij kinderen
> -> shiga toxine producerende [[E. Coli]] (STEC-HUS)
# Kliniek
**= klassieke triade**
- microangiopathische hemolytische [[Anemie]] (Coombs negatief)
	- met aanwezigheid van schistocyten
- thrombopenie
- eind-orgaan schade 
	- [[Acute nierinsufficiëntie]]
	- Encefalopathie
	- [[Epilepsie]]
	- Darmischemie
	- [[Pancreatitis]]
	- [[Hepatitis]]

# Etiologie
## STEC-HUS ("typische" HUS)
= hemolytisch uremisch syndroom door *Shiga toxin-producing [[E. Coli]]*
![[STEC-HUS.png]]
### Kliniek
-> enkele dagen na inname gecontamineerd voedsel
	-> 85% resolutie
	-> 15% klachten -> HUS
		- (bloederige) [[Diarree]] (90%)
		- braken
		- koorts
		- buikpijn

> [!important] Risicofactoren
> - jonge leeftijd (<5j)
> - forse leucocytose (>13000/microliter)

### Behandeling
- vroegtijdig intraveneuze hydratie & adequaat [[Vochtbeleid]]
	- -> minder ernstig verloop
	- -> lagere incidentie oligurie/anurie

> [!important] Vroegtijdige diagnose en doorverwijzing bepaalt **prognose**!

## Atypische HUS
= abnormaliteiten in *complementsysteem*

### Kliniek
-> verhoogd risico op herhaalde TMA episodes
-> symptomen //

### Behandeling 
- vroegtijdige opstart *Eculizumab* (monoclonaal anti-C5 Ab)

## [[TTP]]
= thrombotic thrombocytopenic purpura (<auto-AL tegen het enzym ADAMTS-13)

# Prognose
**30-50% zal op lange termijn blijvende gevolgen hebben**
- [[proteïnurie]]
- [[hypertensie]]
- [[Chronische nierinsufficiëntie]]
- [[Diabetes mellitus]]

-> levenslange opvolging is essentieel
