# Overview

|                                    | Description                                                                                                                       | Start- & endpoint  | Strengths                                                                                                                  | Weaknesses                                                                                                                                                                    | CAVE                                                                 |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- | ------------------ | -------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| [[Case report]]                    | Describe particular clinical phenomenon **in a single patient**                                                                   |                    | - hypothesis generating, formulation of research questions<br>- quick to publish                                           | - cannot establish causality (not to test hypotheses or answer research questions)<br>- limited generalizability (uncle story)                                                |                                                                      |
| [[Case series]]                    | Describe particular clinical phenomenon **in more than one patient with similar problems**                                        |                    | - // to [[Case report]]<br>- can identify patterns<br>                                                                     | - // to [[Case report]]                                                                                                                                                       |                                                                      |
| **Prospective [[Cohort study]]**   |                                                                                                                                   | Exposure → disease |                                                                                                                            | - not good for **rare diseases**<br>- losses to follow-up<br>- not good for diseases with **long latency**<br>- usually **very expensive**<br>- exposure is often not contant |                                                                      |
| **Retrospective [[Cohort study]]** |                                                                                                                                   | Exposure → disease | - better for diseases with long latency than prospective                                                                   | - // to prospective<br>- reliance on past records                                                                                                                             |                                                                      |
| **[[Case-control study]]**         |                                                                                                                                   | Disease → exposure |                                                                                                                            |                                                                                                                                                                               | **No risk**                                                          |
| **Nested case-control**            |                                                                                                                                   | Disease → exposure |                                                                                                                            |                                                                                                                                                                               | - time matched                                                       |
| **Case-cohort study**              |                                                                                                                                   | Disease → exposure |                                                                                                                            |                                                                                                                                                                               |                                                                      |
| **Case cross-over**                |                                                                                                                                   | Disease → exposure | - good for studying **acute outcomes** <br>- eliminates additional cost associated with control group<br>- already matched | - possible **recall [[bias]]**<br>- solid understanding of etiology of disease and predisposing factors is needed<br>- possibly overmatching?                                     | - each case serves as its own control                                |
| **[[Cross-sectional study]]**      | Assess exposure and disease at the same single point in time<br>**= prevalence study**                                            |                    | - usually cheaper and quicker<br>- if designed properly, can estimate disease prevalence in target population              | - temporality (chicken or egg?)<br>- prevalence-incidence [[bias]] (survivorship [[bias]])<br>- latency/exposure misclassification                                                    | - identify **prevalence** rather than incidence (= prevalence study) |
| **[[Ecological studies]]**         | The unit of analysis is a population or group **rather than the individual**, exposure is only assessed for a group or population |                    | - usually easier and cheaper than other designs                                                                            | - ecologic fallacy: the assumption that all members of a group exhibit characteristics of the group at large<br>- exposure misclassification<br>- [[confounders]]                 | - no data on exposure status on each individual in the group         |
